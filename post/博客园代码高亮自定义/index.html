自定义高亮样式

在 高亮Demo 预览喜欢的高亮样式，比如经典的 Monokai

然后到 GItHub 复制对应名字的CSS源码

粘贴到VS Code进行一些修改将 .hljs 替换为 .cnblogs-markdown .hljs ，

如果不想代码换行，还可以添加样式

1
2
3
4
5

pre {
  /*控制代码不换行*/
  white-space: pre;
  word-wrap: normal;
}

将修改后的CSS代码复制到 我的博客-->管理-->设置界面的页面定制 CSS 代码

点击保存，可以测试高亮样式是否生效，如果没有生效，可以给 CSS 代码添加 !important 提升优先级。
